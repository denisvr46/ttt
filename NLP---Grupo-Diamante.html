Anlise de sentimentos aplicado a mensagens do Twitter

Para esse exemplo tentaremos responder a partir de mensagens do Twitter, a seguinte pergunta: Qual o sentimento das pessoas em relao as vacinas?.

Requisitos

  1. Ter uma conta no Twitter.
  2. Estar cadastrado como Twitter Developer.
  3. Acesso a API do Twitter.

Pacotes Utilizados

  • dplyr: Utilizado para manipulao de manipulao de dados.
  • rtweet: Faz a interface de conexo entre o R e a API do Twitter.
  • tm: Possui inmeras funes direcionadas a atividade de minerao de texto.
  • wordcloud: Permite para criar uma nuvem de palavras.
  • syuzhet: Utilizado para classificar os sentimentos. Ele disponibiliza algumas funes teis para a identificao das emoes presentes em textos, entre elas, a funo chamada get_nrc_sentiment() que usa um dicionrio de termos, denominado de NRC Emotion Lexicon, no qual associa palavras emoes e sentimentos, afim de realizar a comparao das palavras e identificar as emoes e sentimentos presentes no texto.

Para fazer a limpeza dos textos podemos utilizar as funes do pacote tm, ou podemos criar as nossas prprias funes

# Funo para limpeza dos textos
limpar_texto <- function(texto) {
  # Convertendo o texto para minsculo
  texto <- tolower(texto)
  # Removendo o usurio adicionado no comentrio
  texto <- gsub("@\\w+", "", texto)
  # Removendo as pontuaes
  texto <- gsub("[[:punct:]]", "", texto)
  # Removendo links
  texto <- gsub("http\\w+", "", texto)
  # Removendo tabs 
  texto <- gsub("[ |\t]{2,}", "", texto)
  # Removendo espaos no incio do texto
  texto <- gsub("^ ", "", texto)
  # Removendo espaos no final do texto
  texto <- gsub(" $", "", texto)
  return(texto)
}
Executando a funo de limpeza de dados

vacina_text <- limpar_texto(vacina_text)

Convertendo os textos em corpus.
O Corpus, so uma coleo de documentos criada pelo R.

vacina_corpus <- VCorpus(VectorSource(vacina_text))

Removendo Stopwords.
Stopwords so palavras que no tenham valor semntico, geralmente so palavras conectivas (com, para, e, a).

vacina_corpus %>% tm_map(removeWords, stopwords("portuguese"))

wordcloud(

Agora transformaremos o corpus em uma matriz de documentos-termos para criarmos um grfico de barras com os termos e sua frequncia.

# Transformando o corpus em matriz de documentos-termos
vacina_doc <-  DocumentTermMatrix(vacina_corpus)

# Removendo os termos menos frequentes
vacina_doc1 <- removeSparseTerms(vacina_doc, 0.97)

# Gerando uma matrix ordenada, com o termos mais frequentes
vacina_freq <- 
  vacina_doc1 %>% 
  as.matrix() %>% 
  colSums() %>% 
  sort(decreasing = T)

# Criando um dataframe com as palavras mais frequentes
df_vacina_freq <- data.frame(
  word = names(vacina_freq),
  freq = vacina_freq)

# Gerando um grfico da frequncia

df_vacina_freq %>%
  filter(!word %in% c("vacina")) %>% 
  subset(freq > 200) %>%
  ggplot(aes(x = reorder(word, freq),y = freq)) +
  geom_bar(stat = "identity", fill='#0c6cad', color="#075284") +
  theme(axis.text.x = element_text(angle = 45, hjus = 1)) +
  ggtitle("Termos relacionados a Vacina mais frequentes no Twitter") +
  labs(y = "Frequncia", x = "Termos") +
  coord_flip()

Realizando a anlise de sentimentos dos tweets.
Para isso utilizaremos a funo get_nrc_sentiment() do pacote syuzhet e passaremos como parmetro os termos da matriz de documentos-termos. Aps obtermos as emoes dos termos, faremos o clculo da frequncia dos sentimentos que utilizaram a #vacina.

# Obtendo os emoes
vacina_sentimento <- get_nrc_sentiment(
  vacina_doc$dimnames$Terms,
  language = "portuguese"
)

# Calculando a frequncia dos sentimentos
vacina_sentimento_freq <-vacina_sentimento %>%
colSums() %>% 
sort(decreasing = T)

# Criando um dataframe com os sentimentos traduzidos, que ser utilizado como converso de domnio. 
sentimetos_traducao <- 
    data.frame(
      sentiment = c(
        "positive",
        "negative",
        "trust",
        "anticipation",
        "fear",
        "joy",
        "sadness",
        "surprise",
        "anger",
        "disgust"
      ),
      sentimentos = c(
        "Positivo",
        "Negativo",
        "Confiana",
        "Expectativa",
        "Medo",
        "Alegria",
        "Tristeza",
        "Surpresa",
        "Raiva",
        "Nojo"
      )
    )

# Tranformando os resultados da frequncia em um dataframe e juntando ao dataframe de traduo
df_sentimento <- 
  data.frame(
    sentiment = names(vacina_sentimento_freq),
    freq = vacina_sentimento_freq
  ) %>% 
  left_join(sentimetos_traducao, by = "sentiment") %>% 
  dplyr::select(-sentiment) %>% 
  arrange(desc(freq))

Visualizando a frequncia dos sentimentos em relao a #vacina

ggplot(data = df_sentimento,
           aes(x = reorder(sentimentos, -freq), y = freq)) +
    geom_bar(aes(fill=sentimentos), stat = "identity") +
    theme(legend.position = "none",
          axis.text.x = element_text(angle = 45, hjus = 1)) +
    xlab("Sentimentos") +
